# RSS-MGR
![Static Badge](https://img.shields.io/badge/Python-3.14-e9f23d?logo=python&logoColor=white)
![Static Badge](https://img.shields.io/badge/PyQt-6.9-41cd52?logo=Qt&labelColor=black)
<br>
RSS-MGR is a simple and lightweight PyQt6 RSS feed editor and features a modern GUI, unlike some local RSS Feed editors.

## ⭐ Features
- Modern UI built on the PyQt6 Framework and Font Awesome icons
- Easy RSS Feed Import from local files and URLs

## ⚙️ Requirements
- `pyqt6`
- `pyqtdarktheme-fork`
- `feedparser`

## 🚀 Running the program
To use this program, simply execute the main.py file directly via python3.
```
python3 main.py
```
